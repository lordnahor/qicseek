***please note that the versions ending in RC are a release candidate and not yet fully tested***

To install any of the following versions:

  • NiscToolset-2008.10.11
  • NiscToolset-2008.10.10
  • NiscToolset-2008.10.10-core(Smaller file used for updating NiscToolset-2008.10.01
  • NiscToolset-2008.10.01
  • NiscToolset-2008.09.12RC
  • NiscToolset-2008.09.12RC-core(Smaller file used for updating NiscToolset-2008.09.09RC
  • NiscToolset-2008.09.09RC

...follow these steps:

  1. Install at least one of the following front-ends:
    1. Visual Studio.NET 2008 (Any of the editions including the free Express Edition from here).
    2. Visual Studio.NET 2005 (Any of the editions including the free Express Edition)
    3. Visual Studio.NET 2003.

    *Run WindowsUpdate (%SystemRoot%\system32\wupdmgr.exe) and make sure you have all .NET framework and Visual Studio Service packs and updates.
    *For more information about how to configure the tools to use the proper Visual Studion version, see FAQ

  2. The NiscToolset itserlf relies on some of the Visual C++ 2008. If you did not install Visual Studion 2008 and plan to use an older version as your frontend, then you need to install the Microsoft Visual C++ 2008 Redistributable Package (x86) (or choose a proper version for you platform from bottom of that page).
  3. You may also need to install "Microsoft Core XML Services (MSXML) 6.0". However, in most cases, this program is already installed on your machine by default. If needed, download and install msxml6.msi from here.
  4. Download the ZIP file of your desired version of NiscToolset (linked on top of this section) and unzip it to any desired directory (e.g. C:\NiscToolset).
  5. Run "<NISCTOOLSET>\NiscEnvironmnet\bin\Install.wsf". Alternatively, you can also simply click on an .nsx file in the <NISCTOOLSET>\NiscDesigns and assign .nsx files to <NISCTOOLSET>\NiscEnvironment\bin\Release\Nisc.exe. You may also need to open <NISCTOOLSET>\NiscEnvironment\bin\Global.config..xml and update the paths of the tools to point to your currect configuration.

To install any of the following versions:

  • NiscToolset-2007.07

...follow these steps:

  1. Install at least one of the following front-ends:
    1. Install Visual Studio.NET 2003. This will also install .NET framework 1.1.
    2. Alternatively you can install Visual Studio.NET 2005.
    3. Alternatively you can install the free Visual C++ 2005 Express Edition from here.

    *Run WindowsUpdate (%SystemRoot%\system32\wupdmgr.exe) and make sure you have all .NET framework and Visual Studio Service packs and updates.
    *For more information about how to use Visual Studion 2005, see FAQ

  2. You may also need to install "Microsoft Core XML Services (MSXML) 6.0". However, in most cases, this program is already installed on your machine by default. If needed, download and install msxml6.msi from here.
  3. Download the ZIP file of your desired version of NiscToolset (linked on top of this section) and unzip it to any desired directory (e.g. C:\NiscToolset).
  4. Run "<NISCTOOLSET>\NiscEnvironmnet\bin\Install.wsf".

To update the NiscToolSet follow these steps:

  1. Download the latest version of NiscToolset ZIP file and unzip it in the same directory that you had already installed the tools (e.g. C:\NiscToolset). Make sure that the files are overwritten.
