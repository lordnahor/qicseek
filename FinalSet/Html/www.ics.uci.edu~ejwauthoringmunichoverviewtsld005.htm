Properties: Value is XML element

  • The value of a property is a well-formed XML (eXtended Markup Language) element

    • XML is a subset of SGML
    • XML specification is being developed by the W3C, is currently very stable, should be a W3C Recommendation later this year.

  • Benefits:

    • i18n support: XML supports ISO 10646 encoding of characters
