Example Lab Assignment

An electron multiplier is a vacuum-tube structure that multiplies incident charges. At each stage of an electron multiplier, each electron, upon striking the plate, generates a Poisson distributed number of electrons for the next stage. Suppose that the mean of the Poisson distribution is 1.2 electrons, and assume that the process runs for 20 generations.
  1. Simulate the electron multiplier process several times, plotting each realization on the same plot. What is the general trend for this process?
  2. Simulate the process 1000 times and calculate the proportion of those 1000 realizations that went extinct. What probability is this proportion estimating?

What to turn into EEE Dropbox

You should turn in two files to EEE Dropbox:

  1. An R script file (extension .R) containing your R code (and comments): Hancock-ExampleLabCode.R
  2. A Word (or pdf or equivalent) document containing R output and answers to the lab assignment questions: Hancock-ExampleLab.doc

The Word document should be all-encompassing -- everything that we need to grade the assignment should be included. The R script file is included so we can run your R code if needed.
