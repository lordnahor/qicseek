During lecture I will periodically stop and ask the class to comment or ask questions. This may result in an uncomfortable silence, but has been scientifically shown to make the class more effective.

 

Components of this course:

  • Reading:
    • Each week will have a reading assignment
    • Readings and lectures will be quizzed.
  • Programming (about 40%):
    • Programming a web crawler
    • Programming a link analyzer
    • Programming a MapReduce algorithm
    • Programming a query scorer
    • Put it all together in a U/I.
    • Programming will be tested be asking you to make modifications to your program during class.
  • Quizzes (about 40%):
    • About every other week in class.
    • Closed book/closed notes.
  • Class Participation (about 20%)A
    • Attendance, Evaluations, etc.

Grade Calibration:

My expectation is that someone who does the work assigned well would receive a B . I occasionally give an A+ and I often find myself giving students C's, D's and F's. Please consider this now if you have particular requirements regarding grade point averages and consider taking an easier course.
