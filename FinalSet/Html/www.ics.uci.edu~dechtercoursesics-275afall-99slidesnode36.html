Starting ECLiPSe in teletype mode

ECLiPSe can be installed in several configurations, according to the installation notes. Enter one of the following commands after the operating system prompt:

  • eclipse: The basic variant of ECLiPSe as described in this manual.
  • peclipse: The parallel ECLiPSe variant as described in chapter .
When one of these commands is invoked, the ECLiPSe system will display the initial header: 
% eclipse
ECRC Common Logic Programming System [sepia opium megalog]
Version 3.5, Copyright ECRC GmbH, Tue Dec 6 12:54 1994
[eclipse 1]:
The list in square brackets specifies the configuration of the running system, i.e. the language extensions that are present. This is followed by the prompt [eclipse 1]:, which tells the user that the top-level loop is waiting for a user query in the module eclipse. The predicate help/0 gives a general help and help/1 gives help about specific built-in predicates.
