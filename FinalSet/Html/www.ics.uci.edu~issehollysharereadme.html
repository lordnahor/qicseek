HollyShare: Limitations of the current version and installation instructions


Limitations (vers. 1/18/02)

  • Current version does not support encryption, all the data are sent in plain text
  • Review system is not implemented, so all its buttons are disabled
  • A couple of other buttons are not implemented yet as well (try all of them to see :-))
  • Resizing of the main window doesn't work properly
  • Check the web later - there may be updates!

Installation procedure

  • Download holly.exe and qt-mt230nc.dll to some directory
  • Run the program
  • Exit the program
  • Edit just created config.ini file:
    • In the first line type your name (probably you need UNIX editor to see the lines properly)
    • Replace the address 0.0.0.0 by the IP address of one of the computers with which you are going to share your files.
  • Run HollyShare on that computer
  • Run HollyShare on your computer
  • Go to UserConfiguration/ModifyShareDirectories and add directories to share
  • Enjoy!
