Assignment #3

 

This assignment is to be conducted in groups of 2, 3 or 4 people.

 

Part 1. (one page)

  • Create a taxonomy of menus which effectively organizes menu types:
    1. Make sure it accounts for the menu types listed in the book,
    2. the menu examples presented in lecture,
    3. and 3 menus types that you identify which have not been discussed.

A discussion of taxonomies can be found here. Don't use the books classification it's not very good or particularly future proof.

 

Part 2. (approximately 5 pages)

  • Compare and contrast MySpace and Facebook in terms of user interfaces
    • Create accounts with both systems if necessary.
    • Read the documentation for the sites, if helpful.
    • Use the sites to see what can be done on both sites.
    • Create a table of features which describe the user interfaces on both sites.
      • Discuss functionality, interface design, and efficiency.
    • Decide which one is better and argue for that perspective using the principles and techniques that we have learned so far in the course.
    • What is your opinion of social networking sites?
      • Based on your stance make a prediction about what the next big social networking site will be.

Compile your work together as a paper document with your group's names on it and submit it by 3/2/2007.
