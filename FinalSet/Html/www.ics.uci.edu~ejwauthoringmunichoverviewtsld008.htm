Property URLs

  • A server may provide a mapping of a property instance to a URL

  • GET retrieves the value of a property

  • DELETE removes a property

  • Benefits:

    • allows use of HTTP methods on properties (as appropriate, e.g., COPY, GET, DELETE, ï¿½)
