Namespace Management: MOVE

  • MOVE method is defined as a COPY followed by a DELETE of the old location

  • All COPY options apply to MOVE as well

  • Benefits:

    • allows the renaming of resources
    • allows positioning the resource in a different location in the hierarchy
    • method is performed without needing to transmit the resource over the wire twice
    • more efficient than separate COPY and DELETE
