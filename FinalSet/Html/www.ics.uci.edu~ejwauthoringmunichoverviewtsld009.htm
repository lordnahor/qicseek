Links using Properties

  • A link is a property

    • XML elements defined in the WebDAV specification are used to describe the source and destination(s) of the link

  • Benefits:

    • treats links like any other descriptive information
    • allows creation of links (relationships) on any content type
    • XML format allows extra descriptive information to be added to a link by using extra tags
