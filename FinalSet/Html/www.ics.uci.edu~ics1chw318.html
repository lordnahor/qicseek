    • What makes the Web so popular?

       The Web, unlike its predecessors, is extremely easy to use. Users do not have to master certain advanced technology to use the Web. In this day and age, it is extremely important for communication and information transfer to be as efficient and as cheap as possible. Vannevar Bush's As We May Think describes the importance of highly-accessible information and communication. The World Wide Web provides exactly this easy access. It is officially defined as a "wide-area hypermedia information retrieval initiative aiming to give universal access to a large universe of documents."

    • What are Web clients and servers, and how do they negotiate data format?

       The client is what a user uses to access the Web. It requests and obtains information from the server. The server in turn searches for the information and delivers the answer of the request from the file that it serves back to the client. The documents or resources are written in Hypertext Markup Language (HTML). The client tells the server what data formats it can handle, and the server delivers the document in the appropriate format. The file name of the document helps the server to identify which format is being used.

    • How does one construct a sentence in Spanish?

      First, you need to know the Spanish vocabulary for the words you want to use in your sentence. For beginners, you can use the small English-Spanish Dictionary. If you already know the simple words, you might want to purchase a good dictionary from the UCI Bookstore. Next, you need to learn the basic verbs, prepositions, adjectives, and other grammar usages. Spanish Lessons will give you a brief idea of how to put different words together using the correct grammar. Practice writing simple sentences. Ask your Spanish-speaking friends to help you.
