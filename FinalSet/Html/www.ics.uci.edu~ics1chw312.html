  1. What is URL?

     URL is a Uniform Resource Locator where you can find the information on the Internet. URL includes a scheme which is the way to retrieve the information, a host where the information is located, and a path which leads you to the particular location of the information. The point is that you can get the information not only from a particular computer but also from any kind of computer on the network. It can also include queries which you will get the answers from it.

  2. What is HTML?

     HTML is a Hyper Text Markup Language which defines the structure of the documents in order to make the documents readable on any computers. So you don't need to create a separate version of the documents, and it makes you easy to access the resources. However, to make the documents into HTML format, you have to follow certain rules. First of all, you have to use tags and entities to create HTML format. HTML format basically consists of HEAD and BODY elements.

  3. Where is the good place to find Japanese culture in Japan?

     There are many places where you can find Japanese culture in Japan. The most famous spot, however, is a city called Kyoto. Kyoto was a Japan's old capital from 794 till 1192. A lot of Japanese culture flourished in Kyoto during that era which is Heian Era. You can find more than 1600 Buddhist temples and 270 Shinto shurines which represent Japan's major religions. Kyoto is very beautiful in spring, fall, and winter, but you might want to avoid summer unles you can stand humidity in Kyoto.
