Compatibility

ECLiPSe 's flexible syntax and the extensive use of its event handling capabilities allows it to build compatibility packages for programs written in other Prolog dialects. Currently packages are provided for C-Prolog, Quintus Prolog SICStus Prolog and the old BSI standard draft. Many of the system parameters can be modified by changing internal flags.
