Description and Objectives

This course considers methods used to study the relationships among several variables. The topics discussed include graphical displays of multivariate data, inference for multiple sample multivariate normal data, data reduction techniques (discriminant analysis, principal components), cluster analysis, and factor analysis.

 

Required Text

Applied Multivariate Statistical Analysis by R. A. Johnson and D. W. Wichern (6th edition). Pearson.

 

Software/Computing

The statistical package R will be used to illustrate examples. R is a free package that can be installed onto machines with different operation systems. For more information about R, please visit http://www.r-project.org. Although R is recommended for your homework assignments, you may also use other packages such as SAS, Splus, or Stata.

 

Grading and Course Requirements

Students will be assigned problem sets, approximately once in each two week period. In addition students will take a midterm examination, and they will have either a final examination, a data analysis project or both. If there is a final examination, it will emphasize data analysis and interpretation more than theory. The grade is determined by performance on this work as follows: homework (approx 30%), exam(s) and/or project (approx 70%).
