  • Project 1: Traffic Check
    • Description: A user-supported real-time traffic reporting site. This system provides an infrastructure for providing traffic reports for local streets automatically by having a cell phone upload location and speed to a server. Additionally it allows a user to overlay traffic accident reports from Yahoo!
