  3. What is a document URL?
    A document URL is a string of characters that distinguish a resource on the internet and identifies the type of media that is being used. It allows for the linking within HTML hypertext read by Web browsers. URLs create information spaces on the Web in terms of the protocol used and the spaces represent a big well of knowledge that the Web can dip its jug in.The basic format for many URLs is : scheme://host:port/path. The scheme is one of the ways to retrieve or send information through the internet. The host is the computer that the resource is located in. The port is a number that distinguishesthe service you are requesting from the host. The path is an the location of a resource on a particular host. URLs & HTML
