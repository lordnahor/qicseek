CHECKIN (1)

  • The CHECKIN method performs the following operations on the Vgraph associated with the Vportal given by the Request-URI, and on the working resource specified in the request body.

    • The working resource is unlocked.
    • The access control for the working resource is set such that no principal has write access (i.e., it is frozen).
    • The server MAY set the version identifier for the working resource to the version identifier specified in the request body.
