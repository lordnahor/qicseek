Experimental Environment



  • Simulator:

    • Enhanced Simscalar 3.0, which incorporates reconfiguration characteristics such as Victim Cache, Stream Buffer, Adaptive Line Size and Fetch Size. Please contact: wtang@ics.uci.edu for more details about it.

  • Compiling Environment:

    • Under SUN Ultra5, using gcc offered in simscalar suite, plus the following flags: -O4.
