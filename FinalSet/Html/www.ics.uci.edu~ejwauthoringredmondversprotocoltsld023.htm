CREATE

  • Creates a Vhandle, Vportal, or combination Vhandle and Vportal resource

  • When creating a Vhandle, the client either:

    • Submits the URI of an existing Vgraph (the Vhandle is for that Vgraph)
    • Submits nothing, and a new Vgraph is created (and the URI of the Vgraph is returned).
