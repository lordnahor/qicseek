System Requirements

The Impromptu Workspace requires Windows XP, Windows Server 2003, or Mac OS X 10.1.1 and later. It uses the built-in WebDAV file system support provided by these operating systems.

Setup Impromptu Workspace

Get Java

Click here to get the latest Java run time environment.

The above link opens a new window. If it asks you to install Java, please follow the instructions. If you already have Java, it will tell you that Java is already installed.

You can also use this Java launcher on Windows to setup Java and Impromptu Workspace.

Windows Setup

  1. If your web browser supports this, you can try open this installer directly in your browser, just like you open a regular exe file.
  2. You can also download the installer file first. Caution: if you have WinZip installed, please choose "Save" when prompted. At the Save As dialog box, choose "All Files" in the Save as type box, and type "install.jar" (please type the quotation marks) in the File name box. Then you can either double click the downloaded file. Or open a command prompt window and type "java -jar install.jar"
  3. Follow the onscreen instructions.
  4. Choose the "Fast Mount" menu item, and click Yes when promputed. You only need to do this once.
  5. Choose the "Impromptu" menu item from the Impromptu program group to open the Impromptu Workspace
  6. To uninstall, choose the "Uninstall Impromptu" menu item from the Impromptu program group.

Mac OS X setup

  1. Download the installer file to the desktop, and then double click the downloaded file.
  2. Follow the onscreen instructions.
  3. Open a Terminal Program
  4. Change to the directory of the Impromptu installation, the default is "/Applications/Impromptu". Type "cd /Applications/Impromptu".
  5. Type "chmod u+x Client Server Peer mountMac" (This makes all scripts executable.)
  6. Start the Impromptu Workspace by typing "./Peer"
