Source Variable Names

  ECLiPSe is able to remember the source names of variables so that e.g. debugging programs becomes much easier. By default, the toplevel loop and the debugger print variables with their source names. The compiler by default checks if the compiled clauses contain singleton variables and emits a warning if it is the case.
