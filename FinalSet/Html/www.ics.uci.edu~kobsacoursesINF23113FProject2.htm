Project 2: Redesign of the UCI Libraries website http://www.lib.uci.edu
The UCI Libraries needs a new site layout that is responsive, i.e. changes according to the screen width of the device viewing it. The task of this project is to
  1. conduct user studies to determine information needs of undergraduates, graduates and faculty (both in their offices and on the go)
  2. plan the layout and the underlying information architecture accordingly
  3. develop mockups for the layout
  4. evaluate the mockups in user studies
The following aspect should be particularly considered:
  • How much information is too much?
  • What language/wording do patrons use when doing research on our pages?  (Librarian vs. Patron terms)
  • How can we make the site more effective?
  • We're currently developing a unified search.  Where should it search?
  • What do you expect to find when using our site?
  • Evaluate the how-do-I page.  Does it provide the information needed? Is the information easy to find and use? What other information would you like to see here?
