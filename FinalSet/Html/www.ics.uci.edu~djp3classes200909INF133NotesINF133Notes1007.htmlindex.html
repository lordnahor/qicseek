Web 3.0
Web 3.0 is basically a reversal of what the focus of web 2.0 was. With 2.0, a vast amount of users contributed small chunks of information to create a massive amount of information, 3.0 will take the massive amount of information available and tailor it to the specific user in a more intuitive, human-like manner
I see Web 3.0 as an extension of Web 2.0 in which content is often user generated but with Web 3.0 users will be able to contribute to the web from all sorts of different devices not restricted to a computer or phone with internet access.
Semantic web
It's a term used to describe the expectations of a new revolution of the world wide web in the digital era. I imagine this revolution to be made up of cloud computing and the use of ubiquitous technologies that act as communicators with the cloud.
