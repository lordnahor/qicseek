MKCOL

  • MKCOL creates a new collection resource at Request-URI

    • If Request-URI exists, MKCOL fails

  • MKCOL may have a request body

    • If no request body is present, the collection created has no members
    • If a request body is present, the server may create new collections, members of those collections, and bodies and properties of those members.
