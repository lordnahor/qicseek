  4. How can one create a page to view in Mosaic? Is a simple thing to do. You need to create your document by using HTML (HyperText Markup Language). This is the language the computer will use to interpret what you are telling it. It requires you to follow specific rules to "tag" or "mark" parts in the document. It may at times be frustrating to follow these rules, one missing tag could distort the document but once you learn them, is easy. Another advantage to usng the HTML form is that it is compatible with any browser so anyone can have access to you file being able to retrieve it from any browser not only Mosaic. To learn how to create a HTML document you can access the James Pollell introduction to HTML. Once you read it and learn how to creat the document then you are ready to make your own home page and access it through Mosaic. a demo document for Mosaic!is available. Once you create a document using the web is twice as fun.
