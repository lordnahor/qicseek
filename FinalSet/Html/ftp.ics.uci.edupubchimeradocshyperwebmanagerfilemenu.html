File menu

The File menu contains the Quit command. The Quit command deactivates the Hyperweb Manager and all other Chimera 2.0 servers on the website. In general, when the Hyperweb Manager is launched, it spawns an instance of the Applet Server and an instance of the Hyperweb Server.

Operations

Quit: This command deactivates the Hyperweb Manager. The Hyperweb Manager will deactivate the Hyperweb Server and any Applet Servers running on the website before it quits.
