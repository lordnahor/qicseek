We see three primary components in building end to end situational awareness technology, namely technology for situational data management, technology for situational data ingest and synthesis from raw input, and tools for analysis and visualization. We are exploring the following challenges:

Event Modeling

Data models for effectively capturing and representing events.

Publications:

U. Westermann, R. Jain. E - A Generic Event Model for Event-Centric Multimedia Data Management in eChronicle Applications. In Proc. of the IEEE Workshop on Electronic Chronicles (eChronicle 2006), Atlanta, Georgia, Apr. 2006. PDF

 

Event Storage

Efficient storage and data management of events, where events are treated as first-class data objects.

Publications:

Event Indexing

Indexing techniques for efficient retrieval of event information. An area we have contributed to is representation and indexing of spatial references in text
