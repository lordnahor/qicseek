Volcanoes on Venus - JARtool experiments

Abstract

The JARtool project was a pioneering effort to develop an automatic system for cataloging small volcanoes in the large set of Venus images returned by the Magellan spacecraft. This package contains a variety of data to enable researchers to evaluate algorithms over the same images as used for the JARtool experiments.
