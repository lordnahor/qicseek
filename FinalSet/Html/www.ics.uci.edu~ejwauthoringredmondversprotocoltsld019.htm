Vgraph (3)

  • A Vgraph has a globally unique identifier, which is a URI (the vgraph: URI scheme is defined for this purpose).

  • All elements of a Vgraph must have globally unique identifiers:

    • Nodes: URIs
    • Arcs: A unique id using the arcid: URI scheme defined in the spec.
