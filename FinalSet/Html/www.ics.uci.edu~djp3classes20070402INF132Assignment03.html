  • Goals
    • The primary purpose of this assignment is to make sure everyone has the capability to put up a web page.
    • The secondary purpose is to introduce XHTML, Javascript, and validation tools in a hands on way, in preparation for the next assignments.
  • Instructions
    • Work with Sylvia, if necessary, to find a place to host a web page. Your own server is fine as long as it is up 24-hours a day.
    • Create a web page which is valid "XHMTL - Transitional". Use the W3C tools to verify that you have no errors. Look at the source of this web page in order to get the correct "DOCTYPE".
    • Your web page should contain the following:
      • a title tag
      • an img tag
      • a list (ordered or unordered)
      • a table
