The goal of our project is to build an infrastructure capable of measuring temperature, sending this data to the Palm PDA via infrared link, collecting and processing them on Palm, and periodically uploading the results to this web page or sending an e-mail.

On this page you see the results of measuring the temperature which were recently uploaded to the server.
 

This project was done for the ICS 213 course (Software for Embedded Systems).

To learn more about the project, download our midterm presentation (on-line presentation or PDF), final presentation (on-line or PDF) or final project report (PDF).
