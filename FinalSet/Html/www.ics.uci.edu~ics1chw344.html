  4. What is a URL?
    My answer is... QQ4 A URL is a Uniform Resource Locater. It is basically an address for a document on the www. The first part of the address defines the protocal. Inother words what type of language is going to be used between the client and server. The next part is the site of the computer. The third part refers to what port the client should connect to the server at. To find out more check out some of these articles in the ics1c reference section.
