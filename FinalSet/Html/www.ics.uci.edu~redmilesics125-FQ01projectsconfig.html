UC IRVINE - ICS 125 PROJECT IN SYSTEM DESIGN

PROJECT DESCRIPTION

We would like to build a configuration management awareness tool that provides on the fly real- time visualizations of what is happening within a configuration management repository. As different developers check out and check in artifacts and make changes, the visualization dynamically updates itself in real-time to graphically show developers the effects of their and others changes in the repository.

The awareness tool should be written in Java, and most likely interface with a wide-area event service from which the events are delivered.
